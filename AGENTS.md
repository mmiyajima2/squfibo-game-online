# AGENTS.md - Claude Code 作業指針

## 最重要ルール
1. **`./docs/game_spec_ja.md` の仕様書を必ず最優先で参照する**
2. **`./docs/socketio_events.md` の仕様書も参照する**
3. *`./docs/online_design.md` を参考情報として参照する*
4. 仕様に記載がない場合や不明点は、実装前に必ず確認を求める
5. セキュリティ・認証に関わる変更は特に慎重に扱う


## 技術スタック
### Client
- **フロントエンド**: Vite + React + TypeScript
- **Linter**: ESLint
- **テスト**: vitest

### Server
- **バックエンド**: expressjs + socket.io
- **データストア**: Redis

## 報告・確認が必要なタイミング
- 仕様書に記載がない機能や判断が必要な時
- 複数の実装方法があり選択が必要な時
- 予期しないエラーや問題に遭遇した時
- 既存コードの大きな変更が必要な時

## エラー対応方針
- エラーログを必ず確認し、根本原因を特定する
- 推測での修正は避け、不明な場合は確認を求める
